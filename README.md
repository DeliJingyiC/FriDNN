# FriDNN
Fricative Detection Using 1D CNNs

## Install

1- Install CUDA and CUDNN, recommended versions

* CUDA Version 10.0

* CUDNN Version 7.4.1

2- Install python (Python >= 3.6 recommended)

3- Setup the required dependencies

```
$ virtualenv env -p python3.6
$ source env/bin/activate
$ pip install -e .
```
